My Projects

"Brewed" UI Project

This project was a creative exploration into UI/UX design and front-end development, bringing an engaging coffee themed interface to life using Angular. The goal was to design an interactive experience where users could explore a curated selection of coffees while showcasing dynamic UI features and modern front-end practices.

The application consumes a public coffee API to display a “Coffee of the Day,” complete with name, description, and flavor profile. I implemented features like randomized coffee selection, smooth scroll triggered animations using Intersection Observer, and dynamic view switching for different coffee details. A custom service layer handles API communication and image validation logic, while the UI emphasizes modern design principles — responsive layouts, animated text sections, and rich visual storytelling. This project demonstrates my ability to transform design concepts into functional, polished interfaces focused on user experience.

PHP Motors

PHPMotors is a full stack car dealership web application built in PHP following an MVC pattern. It uses PDO to communicate with a MySQL back end for dynamic vehicle classifications and CRUD operations on inventory. Users can register, log in, and post reviews; admins can add/edit/delete vehicle records and classifications. The site also handles image uploads and generates thumbnails via the GD library, and features a responsive, include-based template structure (header, nav, footer) for maintainable, consistent styling.
